## Getting Started

### Prerequisites

- Node.js (v18+)
- PostgreSQL

### Installation

1. Clone the repository.
2. Install dependencies for server and client.
   ```bash
   cd server && npm install
   cd ../client && npm install
   ```

3. Start the development servers.
   ```bash
   # Server
   cd server && npm run dev

   # Client
   cd client && npm run dev
   ```
